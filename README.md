### 설명
JPA를 연습하기 위해서 가상의 서점을 만들어보며 연습하는 웹 페이지

### 기능 
1. 회원 : 회원 가입, 회원 목록 조회
2. 상품 : 상품 등록, 상품 목록 조회, 상품 수정
3. 주문 : 주문 등록, 주문 목록 조회, 주문 취소, 주문 삭제

### 기술 스택
1. 백엔드 : JAVA, Spring, JPA, H2 Database
2. 프론트엔드 : Thymeleaf, Bootstrap
